CoffeeSpots

CoffeeSpots is an app for finding the hottest new places to drink coffee. Built using React, the Google Maps/Places APIs are used to generate a map with the hottest coffee spots in a given search area.

Thanks to the HTML5 geolocation and Google geocoding APIs, users can search the map and easily return to their actual location, too.

Danica Discord Bot

Danica is a Discord chat bot that uses the Discord.js API.

This bot makes use of various APIs, such as the Urban Dictionary API (to look up words/phrases), and can also split all the users in a voice channel into two, randomized, even teams.
Github

Simon Game

This is a responsive web "Simon Game" app, built using React, in which the app plays a series of tones and lights and requires a user to repeat the series.

Users can also toggle on a "strict" mode, where when the user fails to successfully repeat a series, is forced to start the game over (rather than at the same "level". The game shows info about the current level, whether strict mode is on, and can be restarted.

Tic Tac Toe Game

Responsive single/two player Tic Tac Toe web game built in HTML, CSS, Javascript & JQuery.

In single player, the game features an unbeatable AI (will force a tie or win every time). In both game modes, player(s) have a choice of either X or O, with X always going first (the AI accounts for the player's choice to ensure that X always goes first).
